Project name : Devdit  - Forum management system

Purpose : The purpose of the Online Forum Management System using MERN Stack is to develop a scalable, user-friendly web platform where users can create, share, and engage in discussions through posts and comments on various topics in a structured, organized manner.

Live Link : https://dev-forum-by-ubaid.netlify.app <br/> 
Server side link : https://github.com/noob-ubaid/devdit-server

Key features :
<ul>
   <h2>User Authentication and Authorization</h2>
  <li>Users can register and log in using email/password or social login like google.</li>
  <li>JWT-based authentication for secure route protection.</li>
  <li>Role-based access (Admin and User).</li>
   <h2>Post Management</h2>
  <li>Users can create, read, and delete posts.</li>
  <li>A user can post 5 posts.if he want to posts more than 5 he has to pay.</li>
  <h2>Comment System</h2>
  <li>Users can comment on posts with a structured comment system.</li>
  <li>Comment truncation with modal ‚ÄúRead More‚Äù functionality.</li>
  <li>Reporting system for inappropriate comments with admin handling.</li>
  <h2>Share Post</h2>
  <li>Users can share the post in Whatsapp and Facebook</li>
  <h2>Voting System</h2>
  <li>Users can upvote or downvote posts to promote valuable discussions.</li>
  <li>Sorting posts by popularity based on vote difference.</li>
  <li>Posts can be filtered and searched based on tags in the banner section for organized discovery</li>
  <h2>Membership System with Stripe Integration</h2>
  <li>Users can purchase membership via Stripe payment gateway.</li>
  <li>Members receive a gold badge and extended posting privileges.</li>
  <h2>User Dashboard</h2>
  <li>Users can view their profile, badges, and recent posts.</li>
  <li>Users can manage their personal posts</li>
  <h2>Admin Dashboard</h2>
  <li>Admins can manage users (promote to admin, view membership status).</li>
  <li>Admins can take actions for reported comments.</li>
  <li>Admins can create and manage announcements for the community.</li>
  <li>Admin can add and manage tags used across posts.</li>
</ul>
